# 📊 AI-Based Project Grading System

An interactive web application built with **Streamlit** to evaluate student projects, assign grades, and generate professional PDF reports.

---

## 🚀 Features
- Enter project and student details
- Input marks for criteria such as:
  - Project Marks
  - Performance
  - Innovation
  - Presentation
- Automatic grade calculation (A–F)
- Pass/Fail detection
- Generates a **PDF report** with:
  - Project details
  - Evaluation table
  - Summary (marks, percentage, overall grade)
  - Remarks
- One-click **download** option for the report

---

## 🛠️ Tech Stack
- **Python**
- **Streamlit** (Frontend & UI)
- **Pandas** (Data Handling)
- **ReportLab** (PDF Generation)
